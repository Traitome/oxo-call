## [Unreleased]

### Bug Fixes

- **response:** Add `deduplicate_flags()` to `sanitize_args()` pipeline — eliminates massive flag duplication from small models (≤3B) that repeat patterns like `--genomeLoad LoadAndKeep --genomeLoad Remove` hundreds of times
- **prompt:** Add explicit "NEVER repeat flags" rule to all three system prompt tiers (full, medium, compact) to prevent flag duplication at generation time
- **ollama:** Use native `/api/chat` endpoint instead of OpenAI-compatible `/v1/chat/completions` for reliable output from all models, especially thinking models (qwen3.5, deepseek-r1)
- **ollama:** Disable thinking mode (`think: false`) for ollama requests to prevent reasoning loop that consumed all tokens without producing content
- **scenario:** Map `--scenario` to ablation flags (`no_skill`, `no_doc`, `no_prompt`) so bare mode truly skips skill/doc/prompt loading
- **qwen3.5:** Add `recommended_max_tokens: 8192` model profile to prevent token exhaustion from thinking overhead
- **streaming:** Handle `reasoning` field in SSE chunks for thinking models that return content via reasoning instead of content

### Performance

- **skill:** Enhance example selection algorithm with parameter matching (3× weight) and subcommand matching (2× boost) for more relevant few-shot examples
