## Concepts

- **Input formats**: bitmapperbs supports common bitmap formats including BMP, GIF, PNG, and TIFF, which are frequently generated by microarray scanners and gel documentation systems.
- **Data model**: The tool processes images as 2D pixel matrices where each pixel represents an intensity value, allowing downstream quantitation of band intensities or spot signals in genomic assays.
- **Output options**: Processed images can be exported in multiple formats with configurable bit-depth (8-bit, 16-bit, 24-bit), compression levels, and metadata preservation.
- **Batch processing**: The tool accepts directories as input, enabling bulk conversion of multiple image files from a single run, which is essential when processing high-throughput experiments.

## Pitfalls

- **Ignoring bit-depth mismatch**: Processing 16-bit gel images and saving as 8-bit output causes irreversible data loss, resulting in truncated quantitation values and inaccurate band intensity measurements.
- **Forgetting color space conversion**: Converting RGB images to grayscale without specifying the correct luminosity algorithm produces unexpected results, affecting downstream image analysis pipelines.
- **Overwriting source files**: Using output paths that match input paths without the `-o` flag permanently destroys original data, making sample recovery impossible.
- **Incorrect image orientation**: Failing to apply rotation or flip flags after acquisition results in misaligned data when comparing multiple gel images or microarray grids.

**Args:** `-i gel_image.tiff -o gel_image.png --format png`
**Explanation:** Converts the gel electrophoresis documentation from TIFF to PNG for easier web sharing while preserving the original pixel data.

**Args:** `-d ./microarray_scans/ -o ./processed/ --format tiff --bit-depth 16`
**Explanation:** Processes an entire directory of microarray scan images to 16-bit TIFF format for quantitative analysis without data loss.

**Args:** `-i spots.png -o spots_resized.png --resize 800x600`

**Args:** `-i rgb_scan.bmp -o gray_scan.bmp --grayscale --method luminosity`
**Explanation:** Converts an RGB microarray scan to grayscale using luminosity weighting, which better reflects human perception of spot intensities.

**Args:** `-i multi_channel.tiff -o red_channel.tiff --extract-channel red`
**Explanation:** Extracts only the red fluorescence channel from a multi-channel gel image for individual band quantitation.
